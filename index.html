📸 Content Production Requirements

Photography Gear

DSLR/mirrorless with macro capability. Tripod for consistent overhead angles. Polarizing filter to cut glare on rope/wood surfaces.

Lighting Standard

Natural light preferred; diffused overcast ideal. For macro detail, portable LED panel with softbox. Consistent white balance (5500K).

Image Processing

Consistent crop ratio (3:2 landscape). Sharpening optimized for web delivery. Compressed WebP/AVIF under 200KB per image.

Total Library Size

Estimated 145–200+ unique photographs. Modular galleries per topic. Lazy-loading implementation required. CDN delivery recommended.
