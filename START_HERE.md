## 🚀 Quick Start

### Development
```bash
cd backend && npm install && npm run dev
cd frontend && npm install && npm run dev
```

### Database
```bash
npx prisma migrate dev
npx prisma db seed
```

### Testing
```bash
npm run test
npm run build
```
